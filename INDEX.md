## üöÄ Quick Commands

### Running a Module
```bash
cd XX-module-name
docker-compose up --build
```

### Building a Module
```bash
cd XX-module-name
mvn clean package
```

### Running Tests
```bash
cd XX-module-name
mvn test
```

### Viewing Logs
```bash
docker-compose logs -f app
```

### Stopping Services
```bash
docker-compose down
```
